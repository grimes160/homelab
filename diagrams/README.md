# Diagrams

Architecture and topology diagrams for the homelab. Sources are kept
as Mermaid (.mmd) so they version cleanly with the rest of the repo
and can be regenerated by any Mermaid renderer.

## Files

- `network-architecture.mmd` - Full physical + logical network topology.
  Shows ISP → modem → router → switching tier → Proxmox host → LXC
  service tier, plus the Tailscale overlay and IoT segment. Edges are
  labeled with link type (1 GbE, PoE, WireGuard) and application flows
  (DNS forwarding, Prometheus scrape paths).

## Rendering

Paste the contents of any .mmd file into https://mermaid.live, or
view inline on GitHub - the platform renders Mermaid in markdown
automatically.
